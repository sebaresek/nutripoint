# NutriPoint ⚡

**NutriPoint** es una plataforma de e-commerce de alto rendimiento dedicada a la venta de suplementos deportivos. El sistema integra un catálogo dinámico de productos, gestión de stock y un flujo de pagos optimizado para deportistas.

## 🌐 Sitio Oficial
Puedes explorar la plataforma aquí: [www.nutripoint.site](https://nutripoint.site/)

## ✨ Características Principales

* **E-Commerce Full Stack:** Gestión integral desde el catálogo de productos hasta la persistencia de datos.
* **Catálogo de Suplementos:** Sistema dinámico para proteínas, creatinas y accesorios.
* **Gestión de Pagos:** Integración de flujos de pago con vistas personalizadas de éxito y error.
* **Arquitectura Escalable:** Separación clara entre el frontend y el servidor de API.
* **Diseño Premium:** Estética moderna, limpia y responsive para una mejor experiencia de compra.

## 🛠️ Stack Tecnológico

### Frontend
* **React.js** (JSX)
* **Tailwind CSS** (UI/UX)
* **Vite** (Build tool)

### Backend
* **Node.js** & **Express**
* **Sequelize ORM** (Modelado de datos)
* **PostgreSQL / MySQL** (Base de datos relacional)
